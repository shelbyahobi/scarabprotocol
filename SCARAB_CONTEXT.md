# SCARAB Protocol - Master Context

## Identity
SCARAB Protocol is a DePIN system connecting real-world ecological hardware to on-chain validation and incentives.

Core principle:
Rewards are based on verified physical activity, not speculation or hash power.

## Stack
- Blockchain: BNB Smart Chain (BSC)
- Contracts: Solidity (UUPS upgradeable where applicable)
- Backend: Node.js / Python
- Hardware: Bokashi, Solar nodes (secure element signing)
- Oracles: Validation + scoring layer
- Frontend: React (Vite)
